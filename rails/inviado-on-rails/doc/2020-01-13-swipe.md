# GraphQL github

```javascript
query {
  search(type:USER, first:100, query:"type:org location:Hamburg") {
    userCount
    edges {
      node {
        ... on Organization {
          id
          databaseId
          createdAt
          updatedAt
          name
          description
          email
          login
          url
          teamsUrl
          websiteUrl
        }
      }
    }
  }
}
```

## with a cursor:

query {

    userCount

          id
          databaseId
          createdAt
          updatedAt
          name
